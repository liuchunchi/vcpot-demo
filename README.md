# vcpot-demo
Demo code for vector commitment based proof of transit

## Usage

- To run the demo, do: 

    `bash run.sh` and follow the instructions.

- To terminate the demo, do: 

    `bash kill.sh` in another terminal.

## Version Requirements
Recommend to use brew for mac
- bash > 4.x
- go > 1.21
- tmux > 3.3
